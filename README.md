# LangChain4j MongoDB Memory

## Overview
A Java implementation for integrating LangChain with MongoDB in memory.

## Documentation
This project provides classes to handle conversation memory management usingMongoDB.

### Features
- In-memory storage
- MongoDB integration
- Simple API for message and conversation management

## Getting Started
To run the application, ensure you have Java 11+ and Maven installed.

### Setup
1. Clone the repository.
2. Configure your MongoDB connection in the `.env` file.
3. Run `mvn clean install` to build the project.
4. Use `java -jar target/langchain4j-mongodb-memory.jar` to start the application.
